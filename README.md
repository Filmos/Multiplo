## Commands
### 〈=ιinner〉
Doesn't do anything on its own, it simply returns its first argument.

#### Examples
##### Input
```
〈=ι123ι234〉
```
 ##### Output
```
123
```

### 〈repιnιcontent〉
Repeats `n` times its `content`.

#### Examples
##### Input
```
〈repι3ιHi,〉
```
 ##### Output
```
Hi,Hi,Hi,
```
